⚠️ PoC: Clickjacking — alts.zone + alts.cool (F-036/037)

Finding: F-036 (alts.zone) / F-037 (alts.cool) · Severity: HIGH
Root cause: No X-Frame-Options or Content-Security-Policy: frame-ancestors header
Impact: An attacker can embed the admin login panel inside an invisible iframe on a malicious page, overlay it with a fake "click here" button, and trick an admin into submitting their credentials to the attacker. Also enables UI redressing attacks on the shop (fake "Buy Now" overlays).

The iframes below confirm both sites can be embedded cross-origin:

alts.cool/login (admin login)

Fix

// Add to every response (nginx.conf):
add_header X-Frame-Options "SAMEORIGIN" always;
add_header Content-Security-Policy "frame-ancestors 'self'" always;

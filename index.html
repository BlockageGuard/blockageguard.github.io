🔴 Critical Findings — Live PoCs

Rate Limit Bypass via X-Forwarded-For — alts.cool Login

The 3-attempt rate limit on customer/admin login is bypassed by sending a random X-Forwarded-For IP on each request. Enables unlimited brute-force with no lockout. Combined with admin panel at /login (no CAPTCHA).

Cross-origin form auto-submission to admin login endpoint processes with no token check or Origin validation. Attacker can force admin browser to authenticate as attacker (session fixation / brute-force relay).

🚨 PoC: CSRF Login Attack — alts.zone Admin Panel

Finding: F-091 / F-092 · Severity: CRITICAL
Target: POST https://alts.zone/admin/session/signin
Root cause: No CSRF token on admin login form + no Origin/Referer validation
Impact: An attacker can embed this page on any domain. When an admin visits, the browser silently sends a cross-origin POST to the admin login endpoint. The server processes it without challenge. An attacker using their own credentials can authenticate the admin's browser session (CSRF login / session fixation), or relay brute-force attempts from the victim's IP (bypassing IP-based lockouts on the attacker's own machine).
